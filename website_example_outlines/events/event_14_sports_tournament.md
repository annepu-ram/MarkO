# Sports Tournament / League / Championship

> Focus: Match schedules, live scores, team profiles, and ticket/registration-driven engagement

## Real Website Examples & Key Insights

| Site | URL | Key Pattern |
|------|-----|-------------|
| Indian Premier League | iplt20.com | Match schedule grid, team cards, live score ticker, stats tables |
| Premier League | premierleague.com | Fixture lists, standings table, player profiles, highlights carousel |
| ESL Gaming | esl.com | Tournament brackets, team rosters, stream embeds, prize pool counters |
| Khelo India | kheloindia.gov.in | Sport categories, medal tally, athlete profiles, registration portal |

**Patterns to incorporate:**
- Match schedule / fixture list with dates, times, venues
- Points table / standings / leaderboard
- Team profiles with rosters and stats
- Live score or match-day countdown
- Ticket booking or team registration as primary CTA
- Highlight reels and photo galleries from past tournaments
- Sponsor/partner logos in ticker format

## Variant A — Cricket / Football League

**Page: Home (Single Page)**

1. **titlebar**
   League logo, nav (Matches, Teams, Standings, Tickets, Gallery, About), button ("Buy Match Tickets").

2. **layout-row (Hero)**
   Left: eyebrow ("Season 5 | April - June 2026") + heading ("Premier City Cricket League") + paragraph ("12 teams. 47 matches. One champion. India's most exciting T20 league returns for its biggest season yet.") + button ("View Schedule") + button ("Get Tickets"). Right: image (action shot of batsman hitting a six or footballer scoring).

3. **br (divider)**
   Visual separator between hero and content sections.

4. **countdown (Next Match Countdown)**
   heading ("Next Match") + caption ("Mumbai Strikers vs Delhi Dynamos | April 12, 7:30 PM | Wankhede Stadium") + countdown timer + button ("Book Seats").

5. **counter-up (League Stats)**
   "12 Teams", "47 Matches", "6 Venues", "500+ Players", "Prize Pool: 5 Crore".

6. **tabs (Match Schedule)**
   Tabs by week or round: Week 1, Week 2, Week 3, etc. Each: columnsgrid (2 columns) with layout-row per match: badge (date) + heading (Team A vs Team B) + caption (time, venue) + button ("Buy Tickets"). Completed matches show: heading (Team A vs Team B) + caption (score) + badge ("Result: Team A won by 5 wickets").

7. **layout-row (Points Table / Standings)**
   heading ("Standings") + columnsgrid representing table rows: each team row with image (team logo) + heading (team name) + caption (Played, Won, Lost, NRR, Points) + badge (on top 4: "Qualified" or "Playoff Contender").

8. **columnsgrid (Teams -- 4 columns)**
   Each: image (team logo/jersey) + heading (team name) + caption (home ground) + paragraph (captain name, key players) + button ("View Squad").

9. **carousel (Match Highlights)**
   Photos and key moments: winning shots, celebrations, crowd reactions, trophy presentations from previous seasons.

10. **layout-row (Star Players)**
   heading ("Players to Watch") + columnsgrid (4 columns): each with image (player action shot) + heading (player name) + caption (team, role) + counter-up (season stat -- runs scored or wickets taken) + badge ("Orange Cap" or "Most Valuable").

11. **ticker (Sponsor Logos)**
    Scrolling logos: title sponsor, team sponsors, broadcast partners, official partners.

12. **layout-row (Fan Zone)**
    Left: heading ("Fan Zone") + paragraph ("Predict match winners, vote for Player of the Match, and win exclusive merchandise.") + button ("Join Fan Zone"). Right: columnsgrid (2 columns): icon + caption ("Match Predictions") + icon + caption ("Fantasy League") + icon + caption ("Win Jerseys") + icon + caption ("Meet & Greet").

13. **form (Ticket Booking)**
    dropdown (select match), dropdown (stand/section: General / Premium / VIP Box), textbox (number of tickets), textbox (name), textbox (email), textbox (phone), button ("Proceed to Payment").

14. **layout-row (Footer)**
    columnsgrid: Column 1 -- league logo + paragraph (about the league). Column 2 -- links (Schedule, Teams, Standings, Rules). Column 3 -- links (Tickets, Fan Zone, Media, Careers). Column 4 -- icon (social links) + caption (media enquiries email) + caption (official broadcaster).

## Variant C — School / College Sports Day / Annual Sports Meet

**Page: Home (Single Page)**

1. **titlebar**
   School/college crest + event name, nav (Events, Schedule, Houses/Teams, Results, Gallery), button ("View Schedule").

2. **layout-row (Hero)**
   Left: eyebrow ("Annual Sports Day 2026") + heading ("Greenfield Academy 28th Annual Sports Meet") + paragraph ("A celebration of sportsmanship, teamwork, and athletic excellence. Join us for two days of thrilling competitions across 25+ events.") + button ("View Events") + button ("Download Schedule PDF"). Right: image (students at a sports day -- running on track, holding trophies, cheering).

3. **br (divider)**
   Visual separator between hero and content sections.

4. **countdown (Event Countdown)**
   heading ("Sports Day Begins In") + countdown timer + caption ("February 20-21, 2026 | School Main Ground").

5. **counter-up (Event Overview)**
   "25+ Events", "4 Houses Competing", "500+ Student Athletes", "2 Days of Competition", "15 Track & Field Events".

6. **columnsgrid (Houses / Teams -- 4 columns)**
   Each: image (house flag/emblem) + heading (house name -- Lions, Eagles, Panthers, Wolves) + caption (house color) + paragraph (house captain name, vice-captain) + counter-up (current points tally) + badge (on leader: "Currently Leading").

7. **tabs (Event Schedule)**
   Tabs: Day 1, Day 2. Each: accordion with time blocks -- "8:00 AM -- Opening Ceremony", "8:30 AM -- 100m Heats (Junior Boys)", "9:00 AM -- Long Jump (Senior Girls)", etc. Each entry: caption (event name, category, venue area) + paragraph (participants per house).

8. **columnsgrid (Event Categories -- 3 columns)**
   Each: icon + heading + caption (number of events). Track Events (sprints, relays, hurdles), Field Events (long jump, high jump, shot put, javelin), Team Sports (football, basketball, cricket, kho-kho, kabaddi).

9. **layout-row (Live Results / Leaderboard)**
   heading ("Points Tally") + columnsgrid (4 columns representing houses): each with image (house emblem) + heading (house name) + counter-up (total points) + progress-bar (visual representation of points) + badge (on leader: "1st Place").

10. **carousel (Past Sports Day Gallery)**
   Photos: races in progress, medal ceremonies, march-past, crowd cheering, teachers participating in fun events.

11. **layout-row (Chief Guest & Inauguration)**
    Left: image (chief guest photo). Right: heading ("Chief Guest") + heading (guest name) + caption (title -- "Olympic Medalist" or "Sports Authority Chairman") + paragraph (brief about the chief guest) + caption ("Inauguration: 7:30 AM, February 20").

12. **layout-row (For Parents & Visitors)**
    heading ("Visitor Information") + columnsgrid (3 columns): each with icon + heading + paragraph. "Seating" (designated parent viewing areas, bring umbrellas for sun), "Parking" (gate 2 parking lot, arrive by 7:00 AM), "Refreshments" (food stalls by school canteen, water stations available).

13. **form (Volunteer / Parent Registration)**
    textbox (parent/volunteer name), textbox (phone), textbox (student name and class), dropdown (role: Spectator / Volunteer / Judge), textarea (any specific requirements), button ("Register").

14. **layout-row (Footer)**
    columnsgrid: Column 1 -- school crest + paragraph (school name and motto). Column 2 -- links (Events, Schedule, Results, Gallery). Column 3 -- caption (school address) + caption (sports coordinator phone) + caption (email) + icon (social links).
