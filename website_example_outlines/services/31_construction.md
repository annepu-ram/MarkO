# Construction — Contractors / Builders / Renovation

> Focus: Trust & Portfolio

## Real Website Examples & Key Insights

| Site | URL | Key Pattern |
|------|-----|-------------|
| Turner Construction | turnerconstruction.com | Project portfolio grid, safety stats, market sector tabs |
| Hensel Phelps | henselphelps.com | Featured project hero, capability breakdown, career CTAs |
| Bechtel | bechtel.com | Mega project showcase, timeline, global presence |
| DPR Construction | dpr.com | Innovation stories, sustainability, VR/tech integration |
| Skanska | usa.skanska.com | Before/after project gallery, safety-first messaging |

**Patterns to incorporate:**
- Large project photography as hero backgrounds
- Before/after project galleries
- Counter-up stats (projects completed, years experience, sq ft built)
- Safety and certification badges prominently displayed
- Process timeline (Design → Permit → Build → Inspect → Handover)
- Client testimonial carousels with project context

## Variant A — General Contractor (Residential & Commercial)

**Page: Home (Single Page)**

1. **titlebar**
   Company name/logo, nav (Services, Projects, About, Process, Contact), badge ("Licensed & Insured"), button ("Get Free Estimate").

2. **layout-row (Hero)**
   Left: eyebrow ("Building Since 2005") + heading ("Quality Construction You Can Trust") + paragraph (residential and commercial projects, on-time delivery) + two buttons ("View Our Work", "Request Quote"). Right: image (active construction site or completed project).

3. **br (divider)**
   Visual separator.

4. **counter-up (Company Stats)**
   "500+ Projects Completed", "18 Years Experience", "50M+ sq ft Built", "98% On-Time Delivery".

5. **columnsgrid (Services — 3 columns)**
   Each: icon + heading + paragraph + button. New Construction, Renovations & Remodeling, Commercial Build-Outs, Additions & Extensions, Concrete & Foundation, Project Management.

6. **tabs (Project Portfolio)**
   Tabs: Residential, Commercial, Industrial. Each: columnsgrid of project images + captions (project name, location, scope).

7. **layout-row (Why Choose Us)**
   Left: heading + paragraph (differentiators). Right: columnsgrid (2x2): icon + heading + caption. Licensed & Bonded, On-Budget Guarantee, Clean Job Sites, Dedicated Project Manager.

8. **layout-row (Our Process)**
   5-step: icon + heading + caption. "Consultation" → "Design & Planning" → "Permits & Approvals" → "Construction" → "Final Walkthrough".

9. **carousel (Client Testimonials)**
   rating + blockquote + caption (client name, project type, location).

10. **ticker (Certifications & Partners)**
    Scrolling logos: OSHA, BBB, local builders association, material suppliers.

11. **accordion (FAQs)**
    How long does a project take?, Do you handle permits?, What's included in the estimate?, Do you offer warranties?, How do payments work?

12. **form (Request a Free Estimate)**
    textbox (name), textbox (phone), textbox (email), dropdown (project type), dropdown (budget range), textarea (project description), button.

13. **layout-row (Footer)**
    Phone, email, office address, license number, service areas, social links.
