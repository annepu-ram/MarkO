# Real Estate — Brokers / Builders / PG/Hostels

> Focus: High Ticket / Trust

## Real Website Examples & Key Insights

| Site | URL | Key Pattern |
|------|-----|-------------|
| Square Yards | https://www.squareyards.com | Search-first hero with filters (buy/rent, city, type, budget), city selector grid, data visualization (price heatmaps), app download push |
| Brick & Bolt | https://www.bricknbolt.com | Cost calculator as primary conversion tool, 4-step process visualization, package comparison cards, ESCROW trust signal, referral program |
| RealEstateIndia.com | https://www.realestateindia.com | Tabbed search (Buy/Rent/PG/Commercial), agent/builder directory, virtual tour listings, mega-footer SEO links |
| Grovy India | https://www.grovyindia.com | Luxury builder with high-res renders, simpler navigation, trust through establishment year (1985), enquiry form focus |
| CommonFloor | https://www.commonfloor.com | Locality guide with infrastructure scores, verified listing badges, statistics bar |

**Patterns to incorporate:**
- Search-first hero is universal for property portals (filters: city, type, budget, BHK)
- City-based navigation is essential for Indian real estate
- Trust signals: Verified badges, RERA numbers, builder credentials, counter stats
- Two-sided marketplace: Both property seekers and sellers/agents need CTAs
- Builders focus on visual quality (renders, galleries), establishment year, enquiry forms
- Heavy footer SEO with extensive locality and city links
- Cost calculators and EMI tools are powerful conversion tools
- Data visualization (price graphs, heatmaps) builds trust on portals

## Variant A — Real Estate Broker / Agency

**Page 1: Home**

1. **titlebar**
   Agency name/logo, nav (Properties, Buy, Rent, About Us, Contact), button ("List Your Property").

2. **layout-row (Hero)**
   Full-width cityscape/property image, overlay: heading ("Find Your Dream Home") + form inline: dropdown (Buy/Rent), textbox (location), dropdown (property type), button ("Search").

3. **counter-up (Agency Credibility)**
   "1,000+ Properties Sold", "500+ Happy Families", "15+ Years", "₹200 Cr+ Transactions".

4. **tabs (Featured Properties)**
   Tabs: For Sale, For Rent, New Projects, Commercial. Each: columnsgrid (3 cols) — image + heading (property name) + caption (location, BHK, sq ft) + counter-up (price) + button ("View Details").

5. **columnsgrid (Property Types — 4 columns)**
   Each: icon + heading + caption. Apartments, Villas, Plots, Commercial Spaces, PG/Hostels, Farmhouses.

6. **layout-row (Why Choose Us)**
   Left: heading + paragraph (expertise, local knowledge, end-to-end support). Right: columnsgrid: icon + caption. RERA Registered, Legal Verification, Loan Assistance, Site Visits.

7. **ticker (Builder Partners)**
   Scrolling builder/developer logos.

8. **layout-row (Recent Transactions)**
   carousel of sold/rented properties: image + badge ("Sold" / "Rented") + caption.

9. **carousel (Client Testimonials)**
   blockquote + rating + caption (buyer/seller, property type).

10. **form (Property Enquiry)**
    textbox (name), textbox (phone), dropdown (looking to: Buy/Rent/Sell), dropdown (property type), textbox (budget range), textbox (preferred location), button.

11. **layout-row (Footer)**
    Office address, RERA number, phone, email, social links.

**Page 2: Property Listing Detail**

1. **carousel (Property Gallery)**
   Multiple property images.

2. **layout-row (Details)**
   heading + badges (BHK, sq ft, floor) + paragraph (description) + columnsgrid (amenities: icon + caption).

3. **form (Schedule Visit)**
   textbox (name), textbox (phone), calendar (visit date), button.
