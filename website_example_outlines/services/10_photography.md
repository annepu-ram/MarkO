# Photography — Studios / Freelancers

> Focus: Visual Portfolio

## Real Website Examples & Key Insights

| Site | URL | Key Pattern |
|------|-----|-------------|
| Zero Gravity | zerogravity.photography | Dark aesthetic, masonry grid, story-driven, Instagram |
| Rajesh Digital | rajeshdigital.com | "Since 1958" heritage, video showreel, tiered pricing |
| A. Rrajani | a-rrajani.com | Personal brand, celebrity roster, category filter bar |
| Portfolio Studio | portfoliostudio.in | Multi-service, filterable gallery, package pricing |

**Patterns to incorporate:**
- Image-first minimal design — photos dominate
- Dark/moody palette for wedding photographers
- Portfolio masonry grid as site centerpiece
- Video showreels for cinematography
- Heritage/legacy as differentiator
- Instagram feed for latest work

## Variant A — Photography Studio (Portraits, Events, Products)

**Page 1: Home**

1. **titlebar**
   Studio name/logo, nav (Portfolio, Services, Pricing, About, Book), button ("Book a Session").

2. **carousel (Hero — Best Work Showcase)**
   Full-width stunning photographs with minimal overlay — just a small watermark logo. Auto-advancing.

3. **br (minimal divider)**

4. **eyebrow + heading**
   "Our Work" eyebrow, "Capturing Moments That Last Forever" heading.

5. **tabs (Portfolio by Category)**
   Tabs: Portraits, Weddings, Events, Products, Fashion, Food. Each tab: columnsgrid (3 cols) of images with hover overlay (caption with shoot details).

6. **layout-row (About the Photographer)**
   Left: image (photographer at work). Right: heading + paragraph (story, style, awards) + badge ("Award-Winning Photographer").

7. **columnsgrid (Services — 3 columns)**
   Each: icon + heading + paragraph + button. Studio Portraits, Wedding Photography, Corporate Events, Product Photography, Fashion Shoots, Photo Restoration.

8. **layout-row (Pricing Packages)**
   3 cards: Basic / Standard / Premium. Each: heading + paragraph (what's included — hours, photos, edits) + counter-up (price) + button ("Book Now").

9. **counter-up (Experience Stats)**
   "500+ Weddings Shot", "10,000+ Portraits", "15+ Years", "50+ Awards".

10. **carousel (Client Testimonials)**
    blockquote + rating + caption (client, event type).

11. **layout-row (Instagram Feed CTA)**
    heading + paragraph + button ("Follow Us on Instagram") + link.

12. **form (Booking Enquiry)**
    textbox (name), textbox (phone), textbox (email), dropdown (service type), calendar (event/shoot date), textarea (details — location, special requests), button.

13. **layout-row (Footer)**
    Studio address, phone, email, social links (Instagram, Pinterest, Facebook).

**Page 2: Full Portfolio**

1. **tabs (All Categories)**
   Full image grids with lightbox-style viewing.

## Variant B — Freelance Photographer (Personal Brand)

**Page: Home (Single Page — Portfolio-Forward)**

1. **titlebar**
   Photographer name, minimal nav (Work, About, Contact), no background — transparent.

2. **layout-row (Hero — Dramatic)**
   Full-width hero image (signature shot), overlay: heading (photographer name) + caption ("Wedding & Lifestyle Photographer").

3. **columnsgrid (Selected Work — Masonry-style 3 columns)**
   Curated images — different aspect ratios, mixed subjects. Each: image + caption (on hover).

4. **layout-row (About)**
   Left: image (self-portrait). Right: heading + paragraph (personal story, shooting philosophy, publications) + link ("Download Media Kit").

5. **ticker (Featured In / Published)**
   Scrolling publication logos where work has appeared.

6. **tabs (Project Stories)**
   2-3 featured projects as tabs. Each: carousel of images + heading + paragraph (the story behind the shoot).

7. **layout-row (Services & Availability)**
   heading + paragraph (what you shoot, availability, travel) + button ("Check Availability").

8. **carousel (Kind Words)**
   Client testimonials — blockquote + caption.

9. **layout-row (Contact)**
   heading + paragraph + form: textbox (name), textbox (email), dropdown (enquiry type), calendar (date if applicable), textarea (tell me about your project), button.

10. **layout-row (Footer — Minimal)**
    Social icons (Instagram, 500px, LinkedIn), email, copyright.

## Variant C — Photo/Video Production Studio

**Page 1: Home**

1. **titlebar**
   Studio brand, nav (Services, Portfolio, Studio Space, Team, Contact), button ("Get a Quote").

2. **video-background (Hero)**
   Behind-the-scenes footage of shoots, overlay: heading ("Where Brands Come to Life") + paragraph + button ("View Our Work").

3. **columnsgrid (Services — 4 columns)**
   Each: icon + heading + caption. Product Photography, Fashion Shoots, Corporate Videos, Social Media Content, Food Photography, 360° Photography, Drone Aerial, Post-Production.

4. **tabs (Portfolio)**
   Tabs: E-Commerce, Fashion, Food, Corporate, Real Estate. Each: image/video grid.

5. **layout-row (The Studio)**
   carousel of studio space images + heading + paragraph (equipment, backdrop options, props, studio size) + button ("Book the Studio").

6. **counter-up (Production Numbers)**
   "10,000+ Products Shot", "200+ Brand Clients", "3 Studio Spaces", "4K Video Ready".

7. **layout-row (How We Work)**
   4-step: icon + heading + caption. "Brief & Concept" → "Pre-Production Planning" → "Shoot Day" → "Edit & Deliver".

8. **ticker (Client Brands)**
   Scrolling brand logos of clients served.

9. **columnsgrid (The Team — 3 columns)**
   Each: image + heading (name) + caption (role: Lead Photographer, Videographer, Editor, Stylist).

10. **carousel (Client Feedback)**
    blockquote + rating + caption (brand, project type).

11. **form (Project Enquiry)**
    textbox (brand name), textbox (contact), textbox (email), dropdown (service), textbox (quantity/scope), calendar (preferred dates), textarea (brief), button.

12. **layout-row (Footer)**
    Studio address, equipment rental info, social media, portfolio download link.
