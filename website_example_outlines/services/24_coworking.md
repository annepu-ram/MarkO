# Coworking — Shared Spaces / Office Rentals

> Focus: Flexibility

## Real Website Examples & Key Insights

| Site | URL | Key Pattern |
|------|-----|-------------|
| Awfis | https://www.awfis.com/ | Booking-first hero (city + type + date), cost calculator vs traditional office, day pass entry point (Rs 300/day), 150+ centres |
| Innov8 | https://www.innov8.work/ | Community-first messaging (events, networking), premium interior photography, customized office builder, Rs 2999/month entry |
| BHIVE Workspace | https://bhiveworkspace.com/ | Bangalore hyperlocal focus, location-first navigation, day pass prominent, enterprise security (biometric, CCTV) |
| Incuspaze | https://www.incuspaze.com/ | Five service lines (Coworking, Managed, Design-Build, Fit-Out, Lease), enterprise-focused, Design & Build portfolio |
| CoFynd | https://cofynd.com/ | Marketplace aggregator model, 100,000+ spaces, brand directory (WeWork, Awfis), coliving alongside coworking |

**Patterns to incorporate:**
- Search/booking interface as hero (city + workspace type + date selection)
- Workspace type comparison (Hot Desk vs Dedicated vs Private Cabin vs Managed Office)
- Location-focused navigation with city and neighborhood detail
- Interior photography of modern, well-designed spaces
- Pricing transparency with "starting from" prices
- Amenities icon grid (WiFi, meeting rooms, pantry, parking, 24/7, power backup)
- Community/events section differentiates from traditional office
- Client logo ticker for enterprise credibility
- Cost calculator/comparison (traditional office vs coworking savings)
- Day pass / low-commitment entry to reduce conversion friction
- Enterprise solutions with separate messaging for corporates
- Inquiry form with team size for customized quotes

## Variant A — Coworking Space / Shared Office

**Page: Home (Single Page)**

1. **titlebar**
   Brand name/logo, nav (Spaces, Pricing, Community, Locations, Contact), button ("Book a Tour").

2. **video-background (Hero)**
   Vibrant coworking footage (people working, collaborating, events), overlay: heading ("Work Anywhere, Grow Everywhere") + paragraph + two buttons ("See Plans", "Book a Tour").

3. **counter-up (Community Numbers)**
   "500+ Members", "3 Locations", "24/7 Access", "100+ Events/Year".

4. **tabs (Workspace Options)**
   Tabs: Hot Desk, Dedicated Desk, Private Office, Meeting Room, Virtual Office, Event Space. Each: image + paragraph (capacity, amenities) + counter-up (starting price/month) + button.

5. **carousel (Space Gallery)**
   Photos of desks, private offices, meeting rooms, lounge, kitchen, phone booths.

6. **columnsgrid (Amenities — 4 columns)**
   Each: icon + caption. High-Speed WiFi, Printing, Coffee & Snacks, Locker, Mail Handling, Parking, Power Backup, Reception.

7. **layout-row (Pricing Comparison)**
   columnsgrid (4 cards): Hot Desk / Dedicated / Private (4-person) / Private (8-person). Each: heading + counter-up (price/month) + paragraph (inclusions) + button.

8. **layout-row (Community)**
   heading + paragraph (networking events, workshops, social hours) + carousel of event photos.

9. **ticker (Member Companies)**
   Scrolling member/startup logos.

10. **layout-row (Locations)**
    tabs or columnsgrid: image + heading (location name) + paragraph (address, metro proximity) + button ("View This Space").

11. **carousel (Member Testimonials)**
    blockquote + rating + caption (member, company).

12. **form (Book a Tour / Enquiry)**
    textbox (name), textbox (email), textbox (phone), dropdown (workspace type), dropdown (team size), dropdown (preferred location), calendar (tour date), button.

13. **layout-row (Footer)**
    All location addresses, phone, email, social, blog link, careers.

## Variant C — Meeting Room / Conference Room Rental

**Page: Home (Single Page — Instant Booking Focus)**

1. **titlebar**
   Brand name, nav (Rooms, Pricing, Locations, FAQ, Contact), button ("Book a Room").

2. **layout-row (Hero)**
   Image (professional meeting room), heading ("Professional Meeting Rooms, On Demand") + paragraph + form inline: dropdown (location) + calendar (date) + dropdown (duration) + button ("Check Availability").

3. **columnsgrid (Room Types — 3 columns)**
   Each: image + heading + caption (capacity) + counter-up (price/hour) + button. Small (2-4), Medium (5-10), Large Boardroom (10-20), Training Room (20-50), Event Space.

4. **tabs (By Location)**
   Tab per location: rooms available, images, amenities, address.

5. **layout-row (All Rooms Include)**
   columnsgrid (4 cols): icon + caption. Projector/Display, Whiteboard, WiFi, Video Conferencing, Coffee/Tea, Stationery, Receptionist, Parking.

6. **layout-row (Pricing)**
   Simple grid: room type × duration × price. caption with "Pay only for what you use".

7. **counter-up (Usage Stats)**
   "10,000+ Meetings Hosted", "4 Convenient Locations", "5-Star Amenities", "Book in 2 Minutes".

8. **carousel (Client Reviews)**
   rating + blockquote + caption (company).

9. **accordion (FAQs)**
   Cancellation policy, AV equipment, food ordering, after-hours access, recurring bookings.

10. **form (Book / Enquire)**
    textbox (name), textbox (email), textbox (phone), dropdown (location), dropdown (room size), calendar (date), dropdown (time), dropdown (duration), textarea (special requirements), button.

11. **layout-row (Footer)**
    Locations list, phone, email, social, corporate packages link.
