# Coaching & Sports — Academies / Swimming / Martial Arts

> Focus: Skill Progress

## Real Website Examples & Key Insights

| Site | URL | Key Pattern |
|------|-----|-------------|
| Bluewater Sports Academy | https://bluewatersportsacademy.com/ | Multi-sport with swimming-first hierarchy, tiered progression (learn to competitive to elite), multi-location support |
| Ramagya Sports Academy | https://ramagyasportsacademy.com/ | Dedicated page per sport for SEO, stats/counter section, age-group targeting (3+ to adult), dual sports + fitness focus |
| Sports Academy of India | https://sportsacademyofindia.com/ | Plan-based pricing (SaaS-like), holistic development (physical + mental), online/offline hybrid model |
| Oriental Martial Arts | https://jaipursportsacademy.com/ | Heritage emphasis ("first gym since 1987"), martial arts primary with multi-sport, health support (physician, dietician) |
| Lakshyan Academy of Sports | https://www.lakshyanacademyofsports.com/ | Infrastructure-first messaging (FINA pool, FIBA courts), residential academy model, Sports Science & Analytics Centre |

**Patterns to incorporate:**
- Hero with action imagery (athletes mid-training, swimming, martial arts stances)
- Program cards with age/skill tiers (Learn, Compete, Elite progression)
- Stats/counter sections (years, students trained, coaches, sports offered)
- Coach profiles with credentials (certifications, competition history, specializations)
- Multi-sport navigation with dedicated page per sport
- Trial class / enrollment CTAs prominent throughout
- Facility gallery with high-quality photos of pools, courts, gyms
- Testimonials from both parents and students
- Age-group segmentation in program navigation

## Variant A — Cricket / Football / Sports Academy

**Page: Home (Single Page)**

1. **titlebar**
   Academy name/logo, nav (Programs, Coaches, Facilities, Achievements, Contact), button ("Enroll Now").

2. **video-background (Hero)**
   Action footage of training sessions, overlay: heading ("Train Like a Champion") + paragraph (professional coaching, world-class facilities) + button ("Join the Academy").

3. **br (divider)**
   Visual separator between hero and content sections.

4. **counter-up (Academy Pride)**
   "500+ Students Trained", "10+ State Players Produced", "15+ Years", "4 Professional Coaches".

5. **columnsgrid (Programs — 3 columns)**
   Each: image + heading + paragraph + badge (age group). Beginners (6-10), Intermediate (11-15), Advanced (16+), Summer Camp, Weekend Program, One-on-One Coaching.

6. **tabs (Training Focus Areas)**
   Tabs: Batting, Bowling, Fielding, Fitness, Mental Conditioning (cricket example). Each: paragraph + image + coach info.

7. **layout-row (Our Coaches)**
   columnsgrid: image + heading (name) + caption (credentials: played for state/national, coaching certifications) + paragraph.

8. **layout-row (Facilities)**
   carousel of facility images + heading + paragraph (turf type, nets, gym, video analysis room).

9. **layout-row (Achievements)**
   ticker of achievements: student name + achievement (selected for state team, tournament wins).

10. **progress-bar (Student Development Path)**
   Visual progression: Beginner → Intermediate → Advanced → Tournament Ready → Selection.

11. **carousel (Parent & Student Testimonials)**
    blockquote + rating + caption (student name, achievement).

12. **layout-row (Fee Structure)**
    columnsgrid (3 cards): Monthly / Quarterly / Annual. Each: heading + paragraph (sessions/week) + counter-up (fee) + button.

13. **form (Enrollment / Trial)**
    textbox (student name), textbox (parent name), textbox (phone), dropdown (age group), dropdown (program), radio (previous experience: Yes/No), button ("Book Free Trial").

14. **layout-row (Footer)**
    Academy address, phone, email, timings, social media.

## Variant B — Swimming Academy / Aquatics Center

**Page: Home (Single Page)**

1. **titlebar**
   Academy name, nav (Programs, Coaches, Pool Info, Schedule, Contact), badge ("Heated Pool"), button ("Register").

2. **layout-row (Hero Split)**
   Left: heading ("Dive Into Excellence") + paragraph (learn to swim, competitive training, all ages) + button ("View Programs"). Right: image (pool / swimmer).

3. **br (divider)**
   Visual separator between hero and content sections.

4. **columnsgrid (Programs — 3 columns)**
   Each: icon + heading + paragraph + badge (level). Learn to Swim (Kids), Learn to Swim (Adults), Competitive Swimming, Water Polo, Aqua Fitness, Lifesaving.

5. **tabs (By Age Group)**
   Tabs: Toddlers (3-5), Kids (6-12), Teens (13-17), Adults (18+), Seniors. Each: program details + schedule + pricing.

6. **layout-row (Our Pool & Facility)**
   carousel of pool images + heading + paragraph (pool dimensions, temperature, changing rooms, safety features).

7. **layout-row (Safety First)**
   columnsgrid (4 cols): icon + caption. Certified Lifeguards, Small Batch Sizes, CPR-Trained Staff, Clean Water Standards, Anti-Slip Flooring.

8. **layout-row (Meet the Coaches)**
   columnsgrid: image + heading + caption (certifications, achievements).

9. **counter-up (Numbers)**
   "3,000+ Students Taught", "100% Safety Record", "Olympic-Size Pool", "National-Level Coaches".

10. **accordion (Class Schedule)**
   By day: time slots, level, coach, availability.

11. **carousel (Student Achievements & Reviews)**
    Medals, competition results, parent testimonials.

12. **layout-row (Pricing)**
    columnsgrid: plan name + sessions + duration + price + button.

13. **form (Registration)**
    textbox (student/swimmer name), textbox (parent name if minor), textbox (phone), dropdown (program), dropdown (level), dropdown (preferred batch), checkbox (can student swim currently?), button.

14. **layout-row (Footer)**
    Address, pool timings, phone, emergency procedures, social.
