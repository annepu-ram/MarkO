# Fitness — Gyms / Yoga Studios / CrossFit

> Focus: Retention & Habit

## Real Website Examples & Key Insights

| Site | URL | Key Pattern |
|------|-----|-------------|
| Gold's Gym India | https://goldsgym.in/ | Bold black/gold palette, gym locator for 150+ locations, before/after transformation stories, franchise model, mobile app promotion |
| cult.fit | https://www.cult.fit/ | Platform/ecosystem (fitness + nutrition + mental health), app-centric, location-based, subscription tiers, vibrant youthful palette |
| Anytime Fitness India | https://www.anytimefitness.co.in/ | 24/7 access as defining feature, global network, no lock-in contract, key fob visual as brand icon, purple brand consistency |
| Habuild | https://habuild.in/ | Online yoga model, free trial as primary conversion, live class schedule display, community/habit messaging, video testimonials |
| FITPASS | https://fitpass.co.in/ | Aggregator model (12,000+ studios), AI trainer feature, per-day cost breakdown, corporate wellness channel |

**Patterns to incorporate:**
- High-energy bold visual design (black, neon, vibrant colors) for gyms
- "Join Now" CTA repeated throughout the page
- Gym/center locator is critical functionality for chains
- Before/after transformation stories are the primary social proof
- Membership tier comparison (Free, Basic, Premium) drives decisions
- Mobile app promotion with QR codes and store badges
- Franchise opportunity sections for gym chains
- Class schedules and program grids for yoga/fitness studios
- Counter stats (members, locations, trainers, years) are universal

## Variant A — Gym / Fitness Center

**Page: Home (Single Page)**

1. **titlebar**
   Gym brand/logo, nav (Programs, Trainers, Membership, Schedule, Contact), button ("Join Now").

2. **video-background (Hero)**
   High-energy gym footage (training, classes, equipment), overlay: heading ("Transform Your Body, Transform Your Life") + paragraph + button ("Start Your Free Trial").

3. **br (wave divider)**
   Visual separator between hero and content sections.

4. **counter-up (Gym Stats)**
   "5,000+ Active Members", "50+ Fitness Classes/Week", "20+ Expert Trainers", "24/7 Access".

5. **columnsgrid (Programs — 4 columns)**
   Each: image + heading + caption. Strength Training, Cardio Zone, HIIT, Functional Training, Personal Training, Group Classes, CrossFit, Zumba.

6. **tabs (Membership Plans)**
   Tabs: Monthly, Quarterly, Annual, Couples/Family. Each: heading + paragraph (access details, perks) + counter-up (price) + badge ("Best Value" on Annual) + button ("Join").

7. **layout-row (Facility Tour)**
   carousel of gym floor, equipment, locker rooms, juice bar + heading + paragraph.

8. **layout-row (Our Trainers)**
   columnsgrid (4 cols): image + heading (name) + caption (certification, specialty) + badge ("Head Trainer").

9. **layout-row (Class Schedule)**
   tabs or accordion by day of week, each showing: time + class name + trainer + duration.

10. **carousel (Transformation Stories)**
   Before/after images + paragraph (member story) + counter-up (weight lost / months).

11. **layout-row (Free Trial CTA)**
    heading + paragraph + form: textbox (name), textbox (phone), dropdown (goal: Weight Loss / Muscle Gain / General Fitness), button ("Claim Free Trial").

12. **ticker (Certifications & Partners)**
    Scrolling: ACE, NASM, supplement brand logos.

13. **layout-row (Footer)**
    Address, hours (24/7 or specific), phone, social media, app download links.

## Variant B — Yoga Studio / Meditation Center

**Page: Home (Single Page — Calm & Mindful)**

1. **titlebar**
   Studio name, nav (Classes, Teachers, Schedule, Workshops, Contact), button ("Book a Class").

2. **layout-row (Hero)**
   Serene image (yoga pose, studio space), heading ("Find Balance, Find Yourself") + paragraph + button ("View Class Schedule").

3. **br (wave divider)**
   Visual separator between hero and content sections.

4. **columnsgrid (Class Types — 3 columns)**
   Each: icon + heading + paragraph. Hatha Yoga, Vinyasa Flow, Yin Yoga, Meditation, Pranayama, Prenatal Yoga.

5. **tabs (Class Levels)**
   Tabs: Beginner, Intermediate, Advanced, Therapeutic. Each: paragraph (what to expect) + button ("Book").

6. **layout-row (Our Teachers)**
   columnsgrid: image + heading + caption (certifications: RYT 200/500, specialty) + paragraph (brief bio).

7. **layout-row (Studio Space)**
   carousel of studio images + heading + paragraph (ambiance, props available, capacity).

8. **accordion (Class Schedule by Day)**
   Each day: table-like layout with time, class name, teacher, level.

9. **layout-row (Pricing)**
   columnsgrid (3-4 cards): Drop-In / 10-Class Pack / Monthly Unlimited / Annual. Each: heading + counter-up (price) + paragraph (validity) + button.

10. **counter-up (Community Numbers)**
   "2,000+ Students Taught", "500+ Classes/Year", "10+ Experienced Teachers", "4.9 Rating".

11. **carousel (Student Testimonials)**
    blockquote + caption + rating.

12. **layout-row (Workshops & Retreats)**
    Upcoming events: image + heading + paragraph + calendar date + button ("Register").

13. **form (First Class Free)**
    textbox (name), textbox (phone), textbox (email), dropdown (class interest), dropdown (experience level), button ("Book Free Class").

14. **layout-row (Footer)**
    Address, hours, phone, social links, newsletter signup.
