# Events — Function Halls / Banquets

> Focus: High Value / Low Frequency

## Real Website Examples & Key Insights

| Site | URL | Key Pattern |
|------|-----|-------------|
| Jade Banquets | jadebanquets.in | Aspirational luxury imagery, venue capacity cards, enquiry form |
| Jaypee Hotels | jaypeehotels.com | Multi-property cards, package pricing, 360° virtual tour |
| WedMeGood | wedmegood.com | Search-first marketplace, real wedding stories, vendor ratings |
| The Oberoi | oberoihotels.com | Ultra-minimal luxury, cinematic photography, floor plans |

**Patterns to incorporate:**
- Enquiry form is primary CTA (not online booking — high-touch sales)
- Venue capacity info on cards helps visitors self-qualify
- Virtual tour / 360° embed reduces need for physical visit
- Gallery is crucial — visitors envision their event
- Event type segmentation: Weddings, Corporate, Social

---

## Variant A — Function Hall / Banquet Hall

**Page: Home (Single Page)**

1. **titlebar**
   Venue name/logo, nav (Halls, Events, Gallery, Pricing, Contact), button ("Check Availability").

2. **carousel (Hero — Venue Showcase)**
   Full-width slides: decorated hall for different events (wedding, reception, corporate), overlay heading + caption (event type) + button ("Book This Hall").

3. **counter-up (Venue Stats)**
   "500+ Events Hosted", "3 Elegant Halls", "Up to 1000 Guests", "15+ Years of Service".

4. **tabs (Our Halls)**
   Tab per hall: image gallery + heading + paragraph (capacity, dimensions, amenities) + badge ("Most Popular") + button ("Enquire").

5. **columnsgrid (Event Types — 3 columns)**
   Each: image + heading + caption. Weddings, Receptions, Corporate Events, Birthday Parties, Engagements, Religious Ceremonies.

6. **layout-row (Amenities)**
   heading + columnsgrid (4 cols): icon + caption. AC, Parking, Catering, DJ/Sound, Decoration, Valet, Bridal Room, Generator Backup.

7. **layout-row (Catering Packages)**
   3 cards: heading + paragraph (menu highlights) + counter-up (price per plate) + button ("View Full Menu").

8. **carousel (Past Events Gallery)**
   High-quality photos of decorated venues, happy clients.

9. **ticker (Client Testimonials)**
   Scrolling review cards: rating + blockquote + caption (event type, date).

10. **layout-row (Virtual Tour CTA)**
    image/video thumbnail + heading + paragraph + button ("Take a Virtual Tour").

11. **form (Enquiry / Check Availability)**
    textbox (name), textbox (phone), dropdown (event type), calendar (event date), textbox (guest count), dropdown (preferred hall), textarea (special requirements), button.

12. **layout-row (Footer)**
    Address with map link, phone numbers, email, social links, cancellation policy link.
