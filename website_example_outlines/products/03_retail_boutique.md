# Retail — Boutiques / Candy Shops

> Focus: Visual Appeal

## Real Website Examples & Key Insights

| Site | URL | Key Pattern |
|------|-----|-------------|
| Neerus | neerus.com | Announcement ticker, mega-menu nav, Instagram feed, trust badges |
| KALKI Fashion | kalkifashion.com | Collection-based merchandising, celebrity section, video lookbook |
| Samyakk | samyakk.com | Showroom virtual tour, worldwide delivery, blog/style guide |
| Bombay Sweet Shop | bombaysweetshop.com | Occasion-based nav, customization, press mentions logo bar |
| Dadu's Sweets | dadus.co.in | Festival-driven merchandising, 2-day delivery, corporate gifting |

**Patterns to incorporate:**
- Announcement ticker bar for promos is standard e-commerce
- Mega-menu navigation for deep product catalogs
- Instagram feed integration drives social proof
- Trust badges bar (Free Shipping, Easy Returns, Secure Payment)
- Occasion-based navigation for gifting businesses
- Festival-driven seasonal collections

## Variant A — Fashion Boutique / Clothing Store

**Page: Home (Single Page)**

1. **titlebar**
   Boutique name/logo, nav (Collections, New Arrivals, About, Visit Us), icon (shopping bag), button ("Shop Now").

2. **carousel (Hero — Lookbook Slideshow)**
   Full-width slides: high-fashion images with overlay heading + caption (collection name, season) + button ("Explore Collection").

3. **br (elegant divider — dashed or wave)**

4. **eyebrow + heading (New Arrivals)**
   "Just In" eyebrow, "New Arrivals" heading centered.

5. **columnsgrid (Product Showcase — 4 columns)**
   Each: image (product photo) + heading (item name) + caption (price) + button ("View Details").

6. **layout-row (Featured Collection Split)**
   Left: large image (styled outfit). Right: eyebrow + heading + paragraph (collection story) + button ("Shop the Look").

7. **ticker (Brand Partners / Labels)**
   Scrolling logos of carried fashion brands.

8. **tabs (Shop by Category)**
   Tabs: Women, Men, Kids, Accessories. Each tab: columnsgrid of product cards.

9. **layout-row (About the Boutique)**
   image (store interior) + heading + paragraph (story, philosophy, founder) + badge ("Est. 2015").

10. **carousel (Customer Photos / Reviews)**
    Customer-submitted outfit photos with rating + caption.

11. **layout-row (Visit Us)**
    Left: image (storefront). Right: heading + paragraph (address, hours) + button ("Get Directions").

12. **layout-row (Footer)**
    Quick links, social media (Instagram prominent), newsletter signup, return policy link.

## Variant B — Candy Shop / Sweet Store

**Page: Home (Single Page — Playful & Colorful)**

1. **titlebar**
   Playful logo, nav (Our Sweets, Gift Boxes, Party Orders, About, Contact), icon (candy/gift).

2. **layout-row (Hero)**
   Full-width image (colorful candy display), overlay: heading ("Sweetness in Every Bite") + paragraph + button ("Order Now").

3. **ticker (Sweet Categories Scroll)**
   Scrolling category cards: icon + caption — Chocolates, Gummies, Lollipops, Traditional Sweets, Sugar-Free.

4. **columnsgrid (Best Sellers — 3 columns)**
   Each: image (candy close-up) + heading (product name) + caption (weight/price) + badge ("Best Seller") + button ("Add to Cart").

5. **tabs (Collections)**
   Tabs: Chocolates, Indian Sweets, Gift Hampers, Sugar-Free, Seasonal. Each: columnsgrid of product cards.

6. **layout-row (Gift Boxes)**
   Left: image (gift box arrangement). Right: heading + paragraph (customizable gift boxes) + button ("Build Your Box").

7. **counter-up (Fun Stats)**
   "10,000+ Happy Customers", "50+ Flavors", "100% Fresh Ingredients", "Same Day Delivery".

8. **carousel (Customer Reviews)**
   Slides: rating + blockquote + caption (customer name).

9. **layout-row (Party & Bulk Orders)**
   heading + paragraph (weddings, birthdays, corporate gifts) + image (party setup) + button ("Enquire for Bulk").

10. **form (Custom Order Request)**
    textbox (name), textbox (phone), dropdown (occasion), textarea (special requirements), calendar (delivery date), button.

11. **layout-row (Footer)**
    Address, delivery areas, social links, FSSAI license badge.

## Variant C — Gift & Stationery Boutique

**Page 1: Home**

1. **titlebar**
   Store name, nav (Gifts, Stationery, Personalized, Corporate, Contact), icon (gift box).

2. **layout-row (Hero Split)**
   Left: heading ("Gifts That Tell a Story") + paragraph + button ("Shop Gifts"). Right: image (curated gift display).

3. **columnsgrid (Shop by Occasion — 4 columns)**
   Each: image + heading + link. Birthday, Anniversary, Wedding, Corporate, Housewarming, Thank You.

4. **layout-row (Personalized Section)**
   image (engraved/personalized items) + heading + paragraph (custom engraving, monogramming) + button ("Start Customizing").

5. **carousel (Trending Products)**
   Product cards: image + heading + caption (price) + badge ("Trending").

6. **tabs (Product Categories)**
   Tabs: Notebooks & Planners, Pens & Writing, Desk Accessories, Wrapping & Cards. Each: product grid.

7. **layout-row (Corporate Gifting)**
   heading + paragraph + counter-up ("500+ Corporate Clients") + button ("Get a Quote").

8. **accordion (FAQs)**
   Delivery, returns, personalization turnaround, bulk discounts.

9. **ticker (Happy Customers)**
   Scrolling testimonial snippets.

10. **form (Contact / Custom Order)**
    textbox (name), textbox (email), dropdown (enquiry type), textarea (message), button.

11. **layout-row (Footer)**
    Store hours, address, Instagram feed link, newsletter signup.
