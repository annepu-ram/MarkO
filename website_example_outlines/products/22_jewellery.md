# Jewellery — Gold / Silver / Imitation Stores

> Focus: Aspiration & Trust

## Real Website Examples & Key Insights

| Site | URL | Key Pattern |
|------|-----|-------------|
| Tanishq | https://www.tanishq.co.in/ | Mega menu navigation, gold exchange program, occasion-based shopping, AI visual search, daily new arrivals |
| Malabar Gold & Diamonds | https://www.malabargoldanddiamonds.com/ | Sub-brands for segments (Mine, Precia, Starlet), lightweight daily wear collection, one-day shipping |
| Kalyan Jewellers | https://www.kalyanjewellers.net/ | Regional wedding positioning, light weight section, Candere e-commerce sub-brand, BIS hallmark badges |
| GRT Jewellers | https://www.grtjewels.com/ | Gift Finder tool, gold savings scheme, GRT Live shopping, men's jewellery section, occasion categorization |
| The Amethyst Store | https://www.theamethyststore.com/ | Niche 92.5 silver focus, traditional South Indian names, curated top picks, clean e-commerce grid |

**Patterns to incorporate:**
- Mega menu navigation with deep hierarchy (by metal, type, collection, occasion, price)
- Hero carousel with large luxurious imagery for collection launches
- Shop by metal type (Gold, Diamond, Silver, Platinum) as primary categories
- Named collections with editorial storytelling
- Occasion-based browsing (Wedding, Daily Wear, Office, Party, Gifting)
- Gold Exchange/Savings Scheme (unique to India)
- Store locator for physical showroom visits (critical for high-value)
- Trust signals: BIS hallmark, certified diamonds, purity guarantee
- Gift Finder tools and vouchers for undecided buyers
- Festival/seasonal campaign banners
