❓ YAML Format Help

YAML Structure

The website is defined using YAML format with clear hierarchy and indentation.

page:
  properties:
    backgroundColor: '#f0f4f8'
    padding: 20

components:
  - name: h1
    properties:
      text: 'Welcome!'
      textAlign: 'center'

Page Properties

Set global page styling at the top level:

  • backgroundColor: Page background color
  • padding: Page padding in pixels
  • margin: Page margin in pixels

Components Array

All components are listed under the components array. Each component has:

  • name: Component type (h1, paragraph, button, etc.)
  • properties: Component-specific settings

Complex Components

Columns Grid:

- name: columnsgrid
  properties:
    count: 2
  columns:
    - components:
        - name: h2
          properties:
            text: 'Left Column'

Tabs:

- name: tabs
  tabs:
    - title: 'Tab 1'
      components:
        - name: paragraph
          properties:
            text: 'Tab content'

Accordion:

- name: accordion
  properties:
    title: 'Click to expand'
  content:
    components:
      - name: paragraph
        properties:
          text: 'Hidden content'

Common Properties

  • text: Display text for headings, paragraphs, buttons
  • color: Text color (hex or color name)
  • backgroundColor: Background color
  • fontSize: Font size in pixels
  • textAlign: left, center, right
  • padding: Inner spacing
  • margin: Outer spacing
  • src: URL for images and videos
  • placeholder: Placeholder text for inputs

YAML Syntax Tips

  • Use 2 spaces for indentation (not tabs)
  • Strings with spaces should be in quotes
  • Arrays use - (dash) prefix
  • Keep consistent indentation levels
  • Use # for comments
